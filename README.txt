This plugin extends WooCommerce to create accounts from WooCommerce checkout to Zoho Contacts through Zoho API.

== Description ==

VLP Create Zoho Account for WooCommerce Customers is used to create accounts to Zoho Accounts CRM from WooCommerce checkout account creation. It is done 
using Zoho CRM Accounts API.

Main Features of the plugin:
- Connecting to Zoho CRM Accounts API using Client ID and Client Secret
- Account creation on Zoho CRM Accounts from WooCommerce checkout

Note: WooCommerce account creation on checkout option should be enabled otherwise there will be no data sent to Zoho.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install VLP WooCommerce Zoho Account to your WordPress Site
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup API keys on 'WooCommerce Zoho Account' menu in Wordpress
